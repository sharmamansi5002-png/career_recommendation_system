# Career Recommendation System

## Project Overview

The Career Recommendation System helps users discover suitable career paths based on their skills, interests, and personality traits.

## Features

* User Registration and Login
* Personality Quiz
* Career Recommendation Engine
* JSON Data Storage
* SQLite Database Integration
* Recommendation History
* Report Generation
* Object-Oriented Programming
* Inheritance and Polymorphism
* Serialization using Pickle

## Technologies Used

* Python
* SQLite
* JSON
* Pickle
